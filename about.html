Kuily's Kitchen

This project is coded in vanilla Javascript, this is my second project and I'm very proud of it. It allows the user to search for a recipee from 'themealdb' API each time the recipe name is typed on the search bar and the search icon button is pressed.



Kuily's Kitchen works both in desktop and mobile versions;

The desktop version has a top navigation menu' where the user can choose between various options included:
  • Shopping list, with all the input fields validated

  • About page where users can find information about this project

    • and finally a:

  • Contacts page where I displayed all my contact details
